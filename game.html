Game Project

January 2018 - present

The Project

After having learned the basics of programming, I had begun looking into the development of video games. In 2017, I discovered Game Maker Studio, a software with it's own built in language that was used for game development. I created several small demo games with the software, simple demos to practice tasks such as creating dialogue boxes or dealing with collisions. After I felt I had learned the basics, I moved on to working on a more fleshed out game.

Alongside my work leading up to this project, I had also been practicing digital art. Pixel art in particular was going to be an important skill for this type of project. I would do small pieces in my spare time and slowly improved to a point I was happy with.

The game itself began with planning. Primarily how the player would move and what type of game it would be at first, with details being fleshed out later. Then I began Programming a basic demo version of the game which would feature one area and showcase a majority of the game's main elements.

The demo game began with simple squares being used for each of the sprites, but has now been updated to use proper pixel sprites with animations for most elements. The demo is nearly complete, with a player character that can move around and fire a small futuristic gun, an animated setting, non player characters that have a few different ways of moving around, a working inventory and item system, a health system, a camera system for viewing the game screen, and a transition system for transporting the player from room to room.

Takeaways

  • Learned about hashmaps and got to apply data structures when building the inventory.
  • Learned ways of implementing common video game elements such as health systems, movement, and collisions.
  • Improved and continue to improve at creating sprites with digital paint software.
  • Gained experience in learning a new programming language.
  • Gained experience in views, scaling and computer graphics.
